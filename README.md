# agent-react-lab
从零实现的最小 ReAct 风格 LLM Agent 实验仓库

A Minimal React-Style LLM Agent Experiment Repository Built from Scratch

---

## 中文说明

本仓库实现一个 **从零构建的最小 ReAct（Reason + Act）风格 LLM Agent**，
用于理解 Agent 的基本运行机制，而非构建完整应用。

实验重点包括：

- 推理-行动（Reason–Act）循环
- LLM 的工具调用（Tool Calling）
- 显式的执行步数控制
- Agent 行为的可解释性

本仓库不包含长期记忆、任务规划或评测模块，
这些内容将在后续实验中逐步引入。

---

## English Version

This repository implements a **minimal ReAct-style LLM agent from scratch**,
focusing on understanding the core execution loop rather than building a full application.

The experiment focuses on:

- Reason–Act execution loop
- Tool calling with LLMs
- Explicit step control
- Interpretability of agent behavior

Long-term memory, planning, and evaluation
are intentionally excluded and will be introduced in later labs.
